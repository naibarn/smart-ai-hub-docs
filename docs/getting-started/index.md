## Key Concepts

Before diving in, familiarize yourself with these core concepts:

### Agents
AI-powered entities that can understand natural language, make decisions, and take actions.

### Workflows
Visual representations of automation logic using nodes and connections.

### MCP (Model Context Protocol)
Universal integration layer that connects to external systems.

### RAG (Retrieval-Augmented Generation)
System that enhances AI responses with relevant context from your data.

## User Roles

Smart AI Hub supports different user roles:

| Role | Permissions | Use Case |
|------|-------------|----------|
| **Admin** | Full access | Workspace management |
| **Developer** | Create & deploy agents | Build automations |
| **User** | Use published agents | End users |
| **Viewer** | Read-only access | Stakeholders |
